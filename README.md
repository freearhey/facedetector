# FaceDetector

Python script for detecting faces in an image.

## Installation

```sh
pip install facedetector-py
```

## Usage

```py
from facedetector import FaceDetector

img = 'path/to/image.jpg' # or NumPy array
faces = FaceDetector.detect(img)

print(faces)
```
